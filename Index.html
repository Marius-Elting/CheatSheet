Git-CheatSheet

^

Auf Github

Schritt 1

Repositorie erstellen

Zuerst clickst du oben rechts auf das kleine Plus neben deinem Profilbild.

Dort wählst du dann "New Repositorie".

Schritt 2

Repositorie bearbeiten

Nun öffnet sich eine neue Seite, hier kannst du nun oben den Wunschnamen eingeben

(Bitte bachte, dass ein Name immer nur einmal vergeben werden kann)

Schritt 3

Kopieren des Import Links

Nun kopierst du dir den Text aus der 2. Import Methode.

In VS-Code:

Schritt 1

in VS-Code navigierst du zu der Datei du die auf Github hochladen möchtest

Schritt 2

Nun Drückst du "Command+J" um das Terminal zu öffnen

Schritt 3

Füge folgendes im Terminal Ein

Nun den kopierten Text von Github einfügen

Nun bist du Fertig
